# AdventOfCode2024

>*Disclaimer:*
>
>*This repository is only for personal/teaching use, the solutions you can find here were written for the puzzles made by [Eric Wastl](https://github.com/topaz) for his amazing yearly online event: [Advent of Code](https://adventofcode.com/). You can find all the original tasks and all the informations about the event [here](https://adventofcode.com/2021).*


In this repository you will find: my code implementation that I used to solve all the puzzles [here]("CodeFolder"), you can find all the ".txt" input files [here]("InputFolder") and all the tests that I run on my code [here]("TestFolder"). Everything was done in Java using IntelliJ as IDE.

In the code implementation folder "*AoC2024.java*" will print out all the solutions to the puzzles, wile the other files (one for each day) have the implementation that solves the puzzles: in each file you will find the code to solve part 1 and part 2 of each puzzle.
